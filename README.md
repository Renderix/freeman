# Freeman

A local voice assistant built with Kotlin Multiplatform. Runs fully on-device — no cloud, no API keys.

- **macOS** — Ollama LLM · Kokoro TTS · Moonshine STT · Silero VAD · OpenWakeWord
- **Android** — Gemma 4 E4B via LiteRT · Kokoro TTS · Silero VAD · OpenWakeWord
